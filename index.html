What is Fluent?

Fluent is a localization system designed to improve how software is translated.

With Fluent, developers can write language-agnostic code and localizers can create natural-sounding translations.

Learn more about Fluent at projectfluent.org.

Fluent in Firefox

Fluent aims to eventually replace all legacy localization formats used in Firefox today. Read the Fluent for Firefox Developers tutorial to learn how to start using Fluent.
