Reusable components in Firefox

Reusable components are web components that have been built to address the issue of having multiple implementations of common UI patterns that get used across Firefox desktop.

These components serve to encapsulate our design system, ensure accessibility and usability across the application, and reduce tech debt.

You can learn more by reading our docs, or checking out our Storybook instance.
